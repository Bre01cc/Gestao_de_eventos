# Gestão de eventos
## Sobre o Projeto
O Sistema de Gestão de Eventos Online é uma aplicação web desenvolvida para facilitar a criação, organização e gerenciamento de eventos. O sistema centraliza as informações e oferece recursos para que organizadores publiquem seus eventos e para que participantes encontrem e acompanhem atividades de seu interesse.

## Descrição Breve
A plataforma possibilita o cadastro de eventos, gerenciamento de categorias, busca e filtragem avançada, além de oferecer, opcionalmente, a compra de ingressos para eventos pagos. O objetivo é simplificar processos, reduzir erros e melhorar a experiência de organizadores e participantes.
